Заполни данные из Firebase Console
1
Зайди на console.firebase.google.com → создай проект
2
Build → Realtime Database → Create Database → выбери регион → test mode
3
Скопируй URL базы: https://ИМЯ-default-rtdb.europe-west1.firebasedatabase.app
4
⚙️ → Project settings → Your apps → </> → скопируй apiKey, appId, projectId
5
В Realtime Database → Rules → вставь ".read": true, ".write": true → Publish
